Quize 1#

Provide the Emmet abbreviation for generating the following HTML DOM structures.

Example Question#

body
|-- header
|-- main
|-- footer

Answer: header+main+footer

Q1#

body
|-- header
|   |-- h1
|   |   |-- "My Store"
|   |-- nav
|-- main
|-- footer

Q2#

nav
|-- ul
|-- |-- li
|   |   |-- a[href="#"]
|   |   |   |-- "Link 1"
|-- |-- li
|   |   |-- a[href="#"]
|   |   |   |-- "Link 2"
|-- |-- li
|   |   |-- a[href="#"]
|   |   |   |-- "Link 3"
|-- |-- li
|   |   |-- a[href="#"]
|   |   |   |-- "Link 4"

Q3#

main
|-- div.store-item
|   |-- a[href="#"]
|-- li*2
    |-- a[href="#"]

Q4#

foot
|-- header
|   |-- nav
|       |-- ul
|           |-- li*3>a
|-- main
|   |-- article
|       |-- h2
|       |-- p*2
|   |-- aside
|-- footer

Q5#

div.store-item
|-- div.container
|   |-- h1.title
|   |-- p.subtitle
|   |-- a.btn[href="#"]
|-- div.overlay

Answers#

  1. header+main+footer

  2. div.container>header>nav+section>h1+p+footer

  3. ul>li>a[href=”#”]+li*2>a[href=”#”]

  4. div.wrapper>header>nav>ul>li3>a+main>article>h2+p2+aside+footer

  5. section.hero>div.container>h1.title+p.subtitle+a.btn[href=”#”]+div.overlay
